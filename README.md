# book loader

## Usage

```bash
node bookloader.js --project ./book-project.json
```

## book-project.json

| Name         | Type    | Description |
|--------------|:-------:|------------:|
| bookName     | String  | Name of the book |
| baseURL      | String  | Base URL |
| debugFiddler | Boolean | Use fiddler proxy do debug |
| bookDir      | String  | Output directory |
